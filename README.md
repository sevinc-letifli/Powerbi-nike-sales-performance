# Powerbi-nike-sales-performance
Nike shoes sales performance and business analytics dashboard developed in Power BI.
# Nike Shoes Satış və Performans Analizi (Power BI)
#  Məqsəd
Bu layihə *Nike* brendinin ayaqqabı satış məlumatlarını analiz etmək, müştəri tələblərini və regional satış trendlərini qiymətləndirmək üçün hazırlanmış interaktiv Power BI hesabatıdır.

## Layihənin Mərhələləri və Görülən İşlər

### 1. Məlumatların Təmizlənməsi (Power Query)
* Xam məlumatlar Power Query-yə daxil edilərək lazımsız sütunlar təmizlənmiş və məlumat tipləri düzəldilmişdir.
* Məhsul qiymətləri və satış miqdarları üzərində düzəlişlər aparılmışdır.

### 2. Məlumat Modeli və DAX (Data Modeling)
* **Star Schema:** Fakt və Dimension cədvəlləri arasında əlaqələr qurularaq sürətli işləyən data model formalaşdırılmışdır.
* **Xüsusi Ölçülər (Measures):** Layihədə biznes nəticələrini ölçmək üçün aşağıdakı DAX measure-ları yaradılmışdır:
  * **Sales (Ümumi Satış)**
  * **Profit (Mənfəət)**
  * **Profit % (Mənfəət faizi)**
  * **# of Orders (Sifariş sayı)**

## Hesabatda Əks Olunan Vizuallar və Səhifələr

### 1. Satışın Ümumi Vəziyyəti (Sales Overview)
* Əsas biznes göstəricilərini əks etdirən interaktiv **KPI Kartları**.
* Zaman oxu üzrə satış trendini göstərən qrafiklər.
* Ən çox satılan Nike ayaqqabı modellərinin analizi.

### 2. Regional və Kateqoriya Analizi
* Satışların regionlar və ölkələr üzrə paylanması.
* Müxtəlif müştəri seqmentləri üzrə (Kişi, Qadın, Uşaq və s. kateqoriyalar) satış həcmləri.

## İstifadə Olunan Texnologiyalar
* **Power BI Desktop**
* **Power Query (ETL)**
* **DAX (Data Analysis Expressions)**

# Nəticə
Bu layihə Power BI istifadə edərək Nike satış məlumatlarının analizini və həmin məlumatlardan biznes üçün dəyərli insight-ların çıxarılmasını nümayiş etdirir. Layihə data vizualizasiya və biznes analitika bacarıqlarımı göstərir.
